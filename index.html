Rainbow jQuery Plguin

Apply colors to a list of DOM objects!

What does this plugin do?

Rainbow walk through a list of DOM objects identified by a simple x-path: $('p').
Each item will be applied with a color fetched from a list (configurable) so you'll obtain a list of different color items.

How do I use this demo?

Just scroll down to examples and click the link to activate them!

Example 01

Click here to apply!

$('#ex01 li').rainbow();

Example 02 - set target property

By default rainbow apply to background property. So let me change this behavior to set the font color:

Click here to apply!

$('#ex02 li').rainbow('color');

Example 03 - customize colors list

Rainbow contains a beautifull list of colors but you can set your own!

Click here to apply!

$('#ex03 li').rainbow({
  colors: [ '#ded4ff', '#b2aacc', '#8f88a3' ]
});
